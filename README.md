_**👦🏻프로젝트 소개👩🏻**_
---  
나 혼자 산다라는 주제에서 알 수 있듯이 남녀노소 구분 없이 홀로 사는 가구자들을 위해 정보들을 제공합니다.  

🏥다이소, 편의점, 병원과 같이 자주가는 장소 검색 도와줍니다.  
🏛️정부, 살고 있는 지역에서 지원하는 혜택 및 정보, 유용한 유튜버들을 선정해 제공합니다.  
🔎사이트 내에서 포털 검색 기능을 제공합니다.  
🛜혼자사는 사용자들이 소통할 수 있는 커뮤니티를 제공합니다.
